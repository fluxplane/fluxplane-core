Use this skill when working on the embedded coder app resources, especially when creating, editing, or reviewing an `agentsdk.app.yaml` app manifest.

Start with `references/configuration.md`. Keep manifests concise, prefer explicit resource names, and validate that referenced agents, sessions, datasources, commands, workflows, operations, connectors, listeners, and channels exist.
